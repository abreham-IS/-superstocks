About SuperStock

SuperStock is a modern, browser-based inventory management system designed specifically for supermarkets and grocery stores. Track products, monitor expiry dates, and keep your shelves fresh — all in one place.

Version 2.0.0 — 2026

Key Features

📦 Product Tracking

Manage your full inventory with production dates, expiry dates, quantities, prices and supplier info.

🔔 Smart Alerts

Automatically flags products that are expiring soon or already expired, with configurable thresholds.

📈 Reports

Interactive charts for stock by category, expiry breakdown, and total inventory value per category.

📂 Import / Export

Backup your data as JSON or CSV. Restore from a JSON file at any time.

🔐 Secure Login

Role-based access with Admin and Manager accounts. Session-based authentication.

⚙️ Customizable

Set your store name, manager name, and alert threshold from the Settings page.
